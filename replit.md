# BCautom - Site Vitrine

## Vue d'ensemble
Site vitrine professionnel pour BCautom, auto-entrepreneur spécialisé en programmation d'automatisme industriel et formation.

## Structure du projet
- `index.html` - Page d'accueil avec présentation des services
- `competences.html` - Détail des compétences techniques (automates, SCADA, réseaux industriels, etc.)
- `wiki.html` - Base de connaissances avec problèmes résolus et solutions
- `styles.css` - Feuille de style CSS pour l'ensemble du site

## Technologies
- HTML5 pur
- CSS3 pur (responsive design)
- Serveur HTTP Python pour le développement

## Objectif
Présenter les services de BCautom en automatisme industriel et créer une base de connaissances technique pour documenter les solutions aux problèmes rencontrés.

## Date de création
Janvier 2025
