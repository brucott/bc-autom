Complément à 2 sur MW

Date : 19/09/2025 |

🔍 Problématique

Les MB et MW ne sont pas signés nativement sur EasySoft (contrairement aux DW).

💻 Code ST

Utiliser le bloc BVxx en mode "NOT" suivi d'une addition :


// Inversion des bits (NOT)
BV01(EN := TRUE, I1 := MW01, QV => MW02);  

// Finalisation du complément à 2
QA05 := MW02 + 1;
