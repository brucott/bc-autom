Hardware / Eaton XV300

Dépannage Tactile XV300 : Écran figé

Date : 22 Avril 2026 | Temps de résolution : 45 min

🔍 Symptômes constatés

L'IHM démarre correctement, mais aucune zone tactile n'est réactive. Le curseur reste fixe même après redémarrage électrique.

[Capture d'écran : Vue du menu système bloqué]

💻 Solution Technique

Nécessite une mise à jour du firmware via l'outil de maintenance Eaton. Voici le segment de configuration CoDeSys à vérifier :


// Vérification de la communication IHM
IF (Status_Touch = FALSE) THEN
    Retry_Count := Retry_Count + 1;
    Reset_Display := TRUE;
END_IF;

✅ Résultat

Après recalibrage forcé via le bouton caché à l'arrière, la dalle est redevenue fonctionnelle. Problème identifié comme une corruption des fichiers de calibration suite à une micro-coupure.
