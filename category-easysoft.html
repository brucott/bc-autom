Mot de passe inattendu

EasySoft 19/09/2025

Description :

Après modification d’un bloc fonction “UF” un mot de passe inexistant est demandé à la réouverture alors qu’il n’en à jamais été saisi.

Solution :

Bug trouvé, réponse de l'Allemagne :

Francois Navarre

Hello Bruno,
The developpment find that when an UFB is writed in ST language (EASYSoft7 or 8) and when the last characktere is a “;” then the EASYsoft 8 had a problem to read this UFB and as result comes a password on it.
When the last charactere is => no problem
Will be corrected in the Version 8.01

Nombre négatif sur MW

EasySoft 19/09/2025

Description :

Les Easy ne transmettent des nombre négatif que sur les DW, les MB et MW ne sont pas signés.

Solution :

Pour cela il suffit d’effectuer un complement à 2 sur le MW. 

BV01 (     
    EN := TRUE,     
    I1 := MW01,     
    I2 := ,     
    ZE => ,     
    QV => MW02 
);  
                                
QA05 := MW02 + 1;

Le bloc fonction BVxx doit être parametré en “NOT”.

A la réception, il sera lu en “signé” si l’appareil est parametré pour ca

First cycle bit

EasySoft 30/10/2025

Description :

Bit à un sur le premier tour de cycle.

Solution :

Ajouter ces lignes en fin de programme 

IF M01 = FALSE THEN     
    M01 := TRUE; 
END_IF;
